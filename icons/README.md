## Design Concept 
The new icon adopts a modern minimalist style, combining the core concepts of "multi-account management" and "security and privacy."

- **Visual Metaphor**:
  - **Stacked Cards**: Symbolize the function of switching and managing multiple accounts.
  - **User Avatar**: Explicitly refers to "account/identity."
  - **Shield Outline (Implicit)**: Conveys a sense of security through the stable stacking of cards.

## Color Guidelines
The icons use a refreshing blue gradient, consistent with the product's main color scheme, conveying professionalism, technology, and trustworthiness.

| Color Name | Hex Code | Use Case |
| :--- | :--- | :--- |
| **Primary Blue** (Primary Color) | `#2563eb` - `#60a5fa` | Gradient for the foreground main card, used to attract visual attention.
| **Secondary Blue** (Secondary Color) | `#bfdbfe` - `#dbeafe` | Gradient for the background secondary card, used to create a sense of depth.
| **White** (White) | `#ffffff` | User avatar and icon body, ensuring high contrast and clarity.
| **Shadow** (Shadow) | `#1e3a8a` (25% Opacity) | Card shadow, adding a sense of three-dimensionality and floating.

## Size and Usage 
Icons are available in multiple sizes to adapt to different display environments, ensuring clear visibility at any resolution.

| Size (px) | Format | Usage |
| :--- | :--- | :--- |
| **16x16** | PNG | Browser toolbar icon, tab icon (Favicon). Simplified design ensures high recognizability at small sizes. |
| **32x32** | PNG | Windows taskbar, high-resolution screen toolbar icon. |
| **48x48** | PNG | Extension management page icon. |
| **128x128** | PNG | Chrome Web Store icon, high-resolution preview icon during installation. |
| **Vector** | SVG | Original vector design for generating export files or marketing materials of any size.

## File Structure
- `icon.svg`: Vector source file
- `icon-16.png`: 16px exported file
- `icon-32.png`: 32px exported file
- `icon-48.png`: 48px exported file
